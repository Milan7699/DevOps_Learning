 // Important git Commands------->
 
 🔹 Repository Setup

git init — Create a new Git repository.
git clone <url> — Copy a remote repository to your local machine.

 🔹 Configuration

git config --global user.name "Name"` — Set your global username.
git config --global user.email "Email" — Set your global email.
git config --list` — Show all Git configuration settings.

🔹 Status & History

git status — Show the current state of files.
git log — Show commit history.
git log --oneline — Show compact commit history.
git diff — Show unstaged changes.
git diff --staged` — Show staged changes.

🔹 Staging & Commit

git add <file> — Stage a specific file.
git add . — Stage all changes.
git commit -m "message" — Commit staged changes.
git commit -am "message" — Stage and commit tracked files.

🔹 Branching

git branch — List all branches.
git branch <name> — Create a new branch.
git checkout <branch> — Switch to a branch.
git checkout -b <name> — Create and switch branch.
git switch <branch> — Switch branches (modern way).
git switch -c <name> — Create and switch branch.
git branch -d <branch> — Delete a branch.

🔹 Merging & Rebase

git merge <branch> — Merge a branch into current branch.
git rebase <branch> — Reapply commits on top of another branch.

🔹 Remote Repositories

git remote -v — Show remote URLs.
git remote add origin <url> — Add remote repository.
git push — Upload local changes to remote.
git push -u origin <branch> — Push and set upstream branch.
git pull — Fetch and merge remote changes.
git fetch — Download changes without merging.

🔹 Undo Changes

git restore <file> — Restore file to last commit.
git restore --staged <file> — Unstage a file.
git reset <file> — Unstage a file (older method).
git reset --hard — Reset working directory completely.
git revert <commit> — Create commit that undoes another commit.

🔹 Stashing

git stash — Save uncommitted changes temporarily.
git stash pop — Restore and remove latest stash.
git stash list — Show all stashes.

🔹 Tags

git tag — List all tags.
git tag <name> — Create a tag.
git push origin <tag> — Push tag to remote.

 🔹 Other Useful Commands

git show <commit> — Show details of a commit.
git blame <file> — Show who changed each line.
git cherry-pick <commit> — Apply specific commit to current branch.
git reflog — Show reference log history.
git clean -f — Remove untracked files.
git help <command> — Show help for a command.
